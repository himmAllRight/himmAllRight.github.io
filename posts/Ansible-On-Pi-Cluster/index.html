Configuring Ansible on the Pi Cluster

By: Ryan Himmelwright
May 10, 2017

In my previous post, I setup my pi cluster and installed the variations of Ubuntu 16.04 Server on each of them. With the cluster setup, I needed an easy way to interact and maintain the systems. This is where Ansible comes in.

Ansible

Ansible is an open source configuration management and automation system, written in Python, and backed by Red Hat. It allows management of groups of computers through the use of modules, standalone units of work (ex: apt, ping, rpm, etc). Ansible is scriptable, via playbooks, a YAML file that can define a set of tasks to orchestrate on a single or group of computers.
